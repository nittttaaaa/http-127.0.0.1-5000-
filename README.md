# Reject Input System (Python)

This project is a simple reject data input system built using Python (Flask) 
to support quality data collection and basic statistical analysis.

## Features
- Input reject data (date, process, problem type, quantity, customer, workcenter)
- Store data in CSV format
- Ready to be integrated with company system (BAAN / ERP)

## Tech Stack
- Python 3.13
- Flask
- CSV file storage

## How to Run
```bash
python reject_input_system.py
